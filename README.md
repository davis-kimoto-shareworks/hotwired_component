# HotwiredComponent
HotwiredComponent adds Hotwired centered generators for Github's ViewComponent

## Usage


## Installation
Add this line to your application's Gemfile:

```ruby
gem 'hotwired_component'
```

And then execute:
```bash
$ bundle
```
